# My CV

This is my curriculum vitae as web developer.

Made with HTML5, CSS 3, [Vue.js](https://vuejs.org) and [Tailwind CSS](https://tailwindcss.org).

## Project Setup

```sh
npm install
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Compile and Minify for Production

```sh
npm run build
```
